# Python Object-Oriented Programming (OOP)
This repository contains lab and exercise solutions for the Python (OOP) course at SoftUni (February 2026).
The goal is to master the fundamental principles of object-oriented design.

### Key Concepts Implemented:
- **Encapsulation**: Managing data access through private and protected attributes
- **Inheritance**: Creating hierarchical structures to reuse code efficiently
- **Abstraction**: Defining blueprints using abstract base classes
- **Polymorphism**: Implementing flexible methods that behave differently based on the object.

### Tech Stack:
- **Language**: Python 3.13.5
- **IDE**: PyCharm
